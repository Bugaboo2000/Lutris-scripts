This document will guide you through the steps of installing Grapejuice on Ubuntu. Please read it in its entirety, from top to bottom.

## Installation options

The follwing Grapejuice installation methods can be used to install Grapejuice on Ubuntu:
- Source Installation
- Package

It is recommended you use the **package** method for Ubuntu. **Do not** install both methods at the same time. Doing so will cause conflicts in your system. If you are not sure which method to use, stick to the recommended option.
